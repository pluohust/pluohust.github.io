硬盘启动过程

Sep 4, 2016 | code

  1. 机器加电

  2. BIOS加电自检

  3. 计算机将硬盘0柱面0磁头1扇区（主引导区MBR）中的512B读入内存0000:7C00处执行（不检测内容）

  4. 检查（WORD）0000:7DFE是否等于0XAA55，若不等于，尝试其他启动介质，若无介质可启动，显示“NO ROM BASIC”然后死机

  5. 跳到0000:7C00处执行MBR中的程序

  6. MBR首先将自己复制到0000:0600处，然后继续执行

  7. MBR在主分区中搜索标志为活动的分区，若无或多个则停止

  8. 将活动分区的第一个扇区读入内存地址0000:7C00处

  9. 检查（WORD）0000:7DEF是否等于0XAA55，若不等于显示”Missing Operating System”，然后停止，或尝试软盘启动

  10. 跳到0000:7C00处继续执行特定系统的启动程序

  11. 启动系统

以上2、3、4、5步由BIOS中的引导程序完成，6、7、8、9、10、11由MBR中的引导程序完成

软盘启动，前5步和硬盘的相同，第6步就直接启动系统了！

参考文献：
卢军，《Linux 0.01内核分析与操作系统》，清华大学出版社
