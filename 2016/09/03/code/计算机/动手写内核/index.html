动手写内核

Sep 3, 2016 | code

引导

物理机开机启动会找到可启动设备（一个魔数决定），然后会将该设备的第一个扇区的内容拷贝到内存（RAM）中。
接着CPU会跳转到内存[0x7c00]，并从[0x7c00]开始执行指令。
以上工作都是Bootloader(grub)要干的活。

内核

Bootloader从内存物理地址[0x100000]的地方加载内核启动。

多重引导规范

grub遵从多重引导规范，其规范为：内核必须在其前8KB字节中包含含头信息（Multiboot header）。这个头信息包含4字节对齐的3个域，分别为：

  • 魔数域：包含魔数[0x1BADB002]。
  • 标志域：如果不关心这个域，置为0。
  • 校验和域：检验和域和前面两个域相加之后的结果必须为0。
