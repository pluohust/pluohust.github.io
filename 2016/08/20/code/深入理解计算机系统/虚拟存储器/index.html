虚拟存储器

Aug 20, 2016 | code

虚拟存储器系统是硬件异常、硬件地址翻译、主存、磁盘文件和内核软件的完美交互，它为每个进程提供了一个大的、一致的和私有的地址空间。

理解虚拟存储器主要关注三个重点：

  1. 虚拟地址和物理地址的翻译和映射过程。虚拟地址使用了分页管理来处理对于物理地址的映射。在从虚拟地址到物理地址的翻译过程，为加速使用了TLB（Translation Lookaside Buffer）,一个小的、虚拟寻址的缓存。
  2. 动态存储器分配。其中的碎片处理是其主要问题。
  3. 垃圾收集。垃圾收集器将存储器视为一张可达图。其中，节点代表堆中的块。不可达节点未垃圾，不能被再次利用。垃圾收集器的角色是维护可达图的表示，并通过释放不可达节点将它们返回给空闲链表 ，以定期回收它们。
